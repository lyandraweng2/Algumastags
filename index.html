Existe algumas tags bem simples no html, são elas:

- <h1/> essa tag é de cabeçalho/título, ma ela não é a unica, já que essa tag faz parte de uma hierarquia, indo até h6.
- <hr/> Essa é bem simples, pq só adiciona uma linha horizontal.
- <p/> Essa é a tag parágrafo, você pode digitar o texto da forma que quiser, mas o texto sempre ficará certo.
- <br/> É responsável por fazer a quebra de linha.
- <img/> Essa tag adiciona imagens no seu site!
